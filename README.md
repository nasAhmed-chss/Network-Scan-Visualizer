# Network Scan Visualizer

Network Scan Visualizer is a web application that visualizes network scan results in an interactive and human-readable format.  
It is designed for cybersecurity learning, analysis, and demonstrations.

The project is built using the Next.js App Router and TypeScript, with a focus on clean UI, scalability, and modern web development practices.

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- React
- Tailwind CSS
- Node.js
